# 🛰️ Universal 5G RF Analyzer
A professional-grade tool for RF Optimization Engineers to analyze Drive Test logs from Actix, Nemo, and DingLi.

## Features
- **Dynamic Mapping:** Automatically detects RSRP, SINR, and Lat/Lon columns.
- **Route Grouping:** Compare multiple files or routes in one dashboard.
- **Interactive Maps:** Color-coded Plotly maps based on RF thresholds.
- **Instant Stats:** Mean, P95, and Min/Max calculations via Polars.

## Deployment
This app is designed to run on **Streamlit Cloud** and connect to a **LINE Bot** via LIFF.
